Hi there, I'm Graham

I'm a 41 year old Australian self-taught front-end web developer living in Chiang Mai Thailand.

I have been living here for two years and I really love the Thai culture and lifestyle and intend to remain living here permanently.

I did a Thai language course for the first six months I was here so I could communicate better with the locals in my village. I found it a very hard language to learn, but I did manage to learn enough to now be able to communicate in simple Thai.

After the language course I started to learn about building websites, I started learning the basics of HTML, CSS and JavaScript mainly from You-tube videos and readings from the internet. I decided I really liked front-end web development so I started to focus on this more by doing some online Udemy courses.

I have now completed the following four online Udemy courses:

  • JavaScript for beginners - Learn JS from scratch – Completed October 2016.
  • The complete web developers course 2.0 – Completed Febuary 2017.
  • Git a web developer job: Mastering the modern workflow – Completed April 2017
  • Web Developers: Build Beautiful Websites - Completed April 2017.

I am very passionate about web development and I enjoy learning about it. I can happily spend hours coding and building, especially building from scratch.

I would really like to start working in the IT industry and as mentioned above, I am particularly interested in front end development.
